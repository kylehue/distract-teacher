# Distract (Teacher Client)
An application for intelligent, real-time student monitoring system.

<img src="https://raw.githubusercontent.com/kylehue/distract-teacher/refs/heads/main/public/distract.png"></img>

### Features
- Real-time student monitoring
- Activity detection and tracking
- Intelligent distraction detection using modern machine learning algorithms
- Live data updates from student clients
- Reports
